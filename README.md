# ninjasite


Database Structure
==================

* **admin**

* **user**/{uid}/
  User private data. No one else has access to here accept for the user.

* **vacancy**/{vacancyId}/
  * _attributes_:
    * url - url of the vacancy

  actual vacancy fetch using Python

* **vacancyFetch**
  * _attributes_:
    * url - url of the vacancy

  experimental fetch using NodeJS
